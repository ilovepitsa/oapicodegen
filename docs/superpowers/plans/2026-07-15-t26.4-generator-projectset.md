# T26.4 — Generator under ProjectSet Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate the generator from `*parser.Document` to `*parser.Project` + `*parser.SchemaIndex`, replacing `g.doc`/`g.modulePath`/`g.features` with `g.project`/`g.schemaIndex`.

**Architecture:** The `Generator` struct stores `*parser.Project` (which contains `Model.Schemas()`, `Paths.Services`, `Paths.Imports`, `Features`) and `*parser.SchemaIndex` (for cross-service ref resolution in T26.5). The `Generate` function signature changes to accept these directly. `main.go` gets a temporary bridge that constructs a `Project` from the parsed `Document` until T26.7 migrates it to `ProjectLoader`.

**Tech Stack:** Go, parser package (T26.1-T26.3), golangci-lint, testify.

```go
func Generate(fw codegen.FileWriter, project *parser.Project, si *parser.SchemaIndex, opts ...Option) error {
	g := &Generator{
		project:     project,
		schemaIndex: si,
		factory:     gogen.NewFileFactory("oapigen"),
	}
	for _, opt := range opts {
		opt(g)
	}

	schemas := project.Model.Schemas()

	if g.project.Features.SplitRequestResponse.Value {
		g.splittable = computeSplittable(schemas)
	}

	for _, sh := range schemas {
		if sh.Name == "" {
			continue
		}

		if err := g.writeSchemaFiles(fw, sh); err != nil {
			return err
		}
	}

	if err := g.writeUTCTimeFile(fw); err != nil {
		return err
	}

	if err := g.writeExpectedValidatorsFile(fw); err != nil {
		return err
	}

	if hasOperations(project) {
		if err := g.writeOperationFiles(fw); err != nil {
			return err
		}
	}

	return nil
}
```

- [ ] **Step 4: Add hasOperations helper**

```go
func hasOperations(project *parser.Project) bool {
	for _, svc := range project.Paths.Services {
		if len(svc.Methods) > 0 {
			return true
		}
	}
	return false
}
```

- [ ] **Step 5: Update computeSplittable to take schemas**

```go
func computeSplittable(schemas []*parser.Schema) map[string]bool {
	out := make(map[string]bool)

	for _, sh := range schemas {
		if sh.Name == "" {
			continue
		}

		if sh.Type == oapiTypeObject && len(sh.Properties) > 0 &&
			len(sh.OneOf) == 0 && len(sh.AnyOf) == 0 && len(sh.AllOf) == 0 {
			out[sh.Name] = true
		}
	}

	for _, sh := range schemas {
		excludeReferencedByComposite(sh, out)
	}

	return out
}
```
